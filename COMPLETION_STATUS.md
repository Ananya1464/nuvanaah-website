## 🎨 Design System

### Color Palette
- **Primary (Gold/Amber):** All CTAs and calls-to-action
  - `bg-amber-500` (default)
  - `bg-amber-600` (hover)
  - `bg-amber-700` (active)

- **Secondary (Sage/Teal):** Calm elements and secondary actions
  - `bg-sage-50` through `bg-sage-700`

- **Neutral (Gray):** Text and structure
  - `text-gray-600` (body text)
  - `text-gray-900` (headings)

### Typography
- **Headings:** Bold (600-700), 1.2 line-height
- **Body:** Regular, 16px, 1.5 line-height
- **Font:** Inter with system fallback
- **Responsive:** Fluid sizing with `clamp()`

### Components
- Buttons (primary, secondary, outline, disabled)
- Cards (standard, feature, product)
- Forms (inputs, labels, validation)
- Modals and dialogs
- Navigation (header, mobile menu, footer)
- Badges and tags

   - Fill in production credentials (Supabase, Razorpay, SendGrid, etc.)

2. **Deploy to Production**
   - Option A: `vercel deploy --prod` (recommended)
   - Option B: `git push origin main` (if auto-deploy configured)

3. **Verify Deployment**
   - Check homepage loads
   - Test product catalog
   - Run checkout flow
   - Verify emails sent
   - Check Sentry (should have no errors)
