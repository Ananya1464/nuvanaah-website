# Nuvanaah Website

Premium medical-grade wigs and hair solutions for cancer patients.

## 🚀 Quick Start

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Build

```bash
npm run build
npm start
```

## 🎨 Features

- **Hero Section**: "Your Crown of Confidence" headline with CTAs
- **Trust Badges**: 100% Real Hair, 45+ Years, Medical Grade, Free Consultation
- **Categories**: Medical Wigs, Premium Wigs, Accessories
- **Newsletter Signup**: Email subscription
- **Responsive Design**: Mobile-first approach
- **Animations**: Smooth Framer Motion animations
- **Premium Colors**: Sage green, gold, and cream palette

## 🛠️ Tech Stack

- **Next.js 14** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **Lucide React** - Icons
